# Sample Files for SignPortal Demo

This directory contains realistic sample files for demonstration purposes of the PNP SignPortal system.

## Directory Structure

```
samples/
├── documents/          # Sample document files (PDF, DOCX, XLSX)
├── attachments/        # Sample attachment files
├── data/              # JSON data files for seeding
└── README.md          # This file
```

## Usage

These sample files can be used to:
1. Test document upload functionality
2. Demonstrate workflow approval processes
3. Show various document types and classifications
4. Seed the database with realistic demo data

## File Types

- **PDF Documents**: Investigation reports, incident reports, intelligence reports
- **Word Documents**: Administrative requests, personnel records
- **Excel Documents**: Financial requests, procurement documents
- **JSON Data**: Pre-configured document metadata for quick seeding

## Notes

⚠️ These are sample/demo files only. Do not use in production environments.
